## 🚀 PROJETO: 
Projeto de APIREST com Java Springboot, usando IDE INTELLIJ.

## 📋 Escola DevSuperior professor Nélio Alves.
Desenvolver um CRUD em uma APIREST com Java + Spring boot seguindo as boas práticas de desenvolvimento de software e utilizando a divisão dos componentes em camadas Repository, Service e Controller. 
Também fiz uso do padrão 'DTO - Data Transfer Object' para transformar objetos em objetos DTO.

Java / IntelliJ IDEA / Spring / Spring boot / Spring web / Sppring database H2 / Maven / JPA
